GET
/sync/all-products-from-apis
Dispara o sync manual completo (Omnia → WooCommerce)
▾

Executa a mesma lógica do cron automático (a cada 30 minutos). Cria produtos novos, atualiza os que mudaram (preço, estoque, nome, atacado) e move para rascunho os que não existem mais no Omnia.

Atenção: pode demorar vários minutos dependendo do volume. Não chame em paralelo com o cron.

POST
/sync/woocommerce/webhook/created-order
Webhook de novo pedido — cria o pedido no Omnia
▾

Headers obrigatórios

HeaderDescrição
x-wc-webhook-signatureAssinatura HMAC SHA-256 gerada pelo WooCommerce obrigatório
x-wc-webhook-topicDeve ser order.created obrigatório

Fluxo

1. Valida assinatura HMAC com WC_CREATED_ORDER_WEBHOOK_SECRET
2. Busca código IBGE pelo CEP do cliente
3. Se o cliente não existe no Omnia, cria automaticamente
4. Cria o pedido no Omnia com itens, pagamento e dados de entrega

Configurar no WooCommerce: WooCommerce → Configurações → Avançado → Webhooks → Tópico: Pedido criado.
